# v2.21.0 版本说明

## 功能增强 (Features)

- **req 插件**：`/req:review-pr` 无阻塞问题时，检查是否有审核人并提示提交审核通过状态 ([409ca72](https://github.com/zhouhao4221/devflow-claude/commit/409ca72))
  - 有审核人时提示「是否在平台提交审核通过状态？」，调用 Gitea/GitHub API 将 Review 状态更新为 Approved
  - 无审核人时仅展示无阻塞结果，引导使用 `/req:review-pr merge`
  - 审核通过与合并严格分离，不再提供「审核并合并」选项

## 问题修复 (Bug Fixes)

- **req 插件**：`/req:review-pr` 审核通过 API 调用加返回值校验，失败时明确报错并提示原因 ([690870b](https://github.com/zhouhao4221/devflow-claude/commit/690870b))
- **req 插件**：`/req:review-pr` 提示语明确「审核通过 ≠ 合并，PR 仍处于 Open 状态」([59ae34a](https://github.com/zhouhao4221/devflow-claude/commit/59ae34a))
- **req 插件**：`/req:review-pr` 去除「审核并合并」选项，审核与合并操作严格分离 ([7d17163](https://github.com/zhouhao4221/devflow-claude/commit/7d17163))
