# v2.8.0

> 发布日期：2026-04-16
> 版本范围：v2.7.1..HEAD

## 新功能

- **SessionStart 新用户两步启动引导**（req 3.4.0）：`session-context.sh` 检测到仓库未初始化时输出欢迎引导，提示新用户执行 `/req:init` + `/req:branch init` 两步完成启动；已绑定项目但未配置 `branchStrategy` 时，在上下文末尾追加 ⚠️ 提示，引导执行 `/req:branch init`；完成配置后提示自动消失
- **多语言帮助文档（中 / 英 / 韩）**：
  - 新增 `README.en.md`、`README.ko.md` 和 `docs/tutorial.en.md`、`docs/tutorial.ko.md`，各文件顶部加语言切换链接
  - `plugins/api/docs/` 抽出三语教程文件（`tutorial.zh/en/ko.md`）
  - `/req:help`、`/api:help`、`/pm:help` 全部支持 `--lang=zh|en|ko` 参数
  - 默认读取 `.claude/settings.local.json` 的 `language` 字段，未设置时回退中文

## 优化

- README 快速开始改为「两步启动」：`/req:init <project-name>` + `/req:branch init`，与 SessionStart 引导保持一致
- `docs/tutorial.md` 开头加 TL;DR 说明 SessionStart 引导流程，1.4 分支策略章节由「可选」改为「强烈推荐」
- `CLAUDE.md` SessionStart Hook 描述补充两种新分支行为（未初始化仓库、已绑定但未配置分支策略）
- README 三语版本号统一到当前实际 plugin.json 版本

## 版本升级

- marketplace：2.7.1 → 2.8.0
- req：3.3.1 → 3.4.0
- api：0.2.2 → 0.3.0
- pm：0.1.2 → 0.2.0
