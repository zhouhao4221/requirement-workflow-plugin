# v2.3.2

> 发布日期：2026-04-14
> 版本范围：v2.3.1..HEAD

## 修复

- `/req:release` 的 PostToolUse hook 改从 stdin JSON 读取文件路径，修复 `$CLAUDE_FILE_PATH` 失效导致缓存同步不触发
  - `sync-cache.sh` 和 `validate-requirement.sh` 改用 `jq` 解析 `tool_input.file_path`
  - `hooks.json` 移除不存在的 `$CLAUDE_FILE_PATH` 参数
- `sync-cache.sh` 恢复位置参数回退，兼容未注入 stdin 的调用方式
- `/req:release` draft 模式 `target_commitish` 改为绑定主分支名，避免 Release 创建在错误分支

## 重构

- 移除 `/req:release` 的 prerelease 概念，统一按分支策略判定发布类型

## 文档

- req 插件新增 Memory 隔离规则，禁止模板化命令受 auto-memory 影响
